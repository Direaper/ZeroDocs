Creating an object is a common action within Zero Engine, in this page we outline all the ways to create one (using Sprite as an example).



 # `Create  > Create Sprite`

 ![CreateSprite](https://media.githubusercontent.com/media/zeroengineteam/ZeroFiles/master/doc_files/46945.gif)
